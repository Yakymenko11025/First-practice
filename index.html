Чому саме у нас?

На спеціальності "інженерія програмного забезпечення" ЧНУ ім.П.Могили склався унікальний мікроклімат для отримання навичок з розробки програмного забезпечення, утворений комбінацією наступних факторів

Про кафедру

Кафедру інженерії програмного забезпечення було створено у 2016 році у результаті реорганізації кафедри інтелектуальних інформаційних систем (історія якої у свою чергу бере початок у 2003 році) шляхом її поділу на дві випускаючі кафедри. Завідувач кафедри – канд. техн. наук, доцент Давиденко Євген Олександрович.

Акредитація освітньої роботи

У 2019 році кафедра пройшла акредитацію освітньої програми за спеціальністю 121 «Інженерія програмного забезпечення» першого рівня вищої освіти.

Популярність професії та високий рівень заробітної плати

Професія програміста з кожним роком стає все більш популярною та високо оплачуємою. Високо кваліфіковані фахівці в області розробки програмного забезпечення затребувані як В Україні, так і за кордоном.

Завдяки своєму професійному рівню, наші студенти мають можливість працювати за спеціальністю, набуваючи професійний досвід, ще продовжуючи своє навчання на старших курсах, а після випуску є затребуваними та високооплачуваними спеціалістами своєї професійної області.

Кафедра інженерії програмного забезпечення забезпечує надання ґрунтовних знань з мов програмування та технологій:

Для WEB-розробників (технології / посада):

  • JavaScript (SPA frameworks), HTML5, CSS3 / Frontend Developer
  • PHP / Backend Developer
  • Spring / Java Developer
  • Django / Python Developer
  • NET MVC / C# Developer
  • MySQL / Database Developer
  • UI Developer

Для Mobile-розробників (мова програмування / посада):

  • Android API / Android Developer
  • Objective C, Swift / iOS Developer

Для Embedded System розробників (мова програмування, засоби проектування):

  • С / C++ V.11
  • Assembler
  • VHDL
  • Середовища розробки: AVR Studio, mbed, Multisim, Microcap, LabView, Altium Designer, System Work Bench for STM32, MPLab, ARDuino IDE
  • Системи автоматизованого проектування (САПР): Altium Designer, AutoCAD, P-CAD
  • Системи комп’ютерної математики (СКМ): Maple, MATLAB

Для Desktop-розробників (технологія / посада):

  • 1C Підприємство / 1C Developer
  • Java FX / Java Developer
  • Python / Python Developer
  • .NET, WPF / C# Developer

Для здійснення підготовки висококваліфікованих фахівців та реалізації навчально-дослідних програм, факультет КН забезпечено сучасною комп’ютерної технікою та обладнанням:
  • 12 комп’ютерних класів, оснащених сучасною комп’ютерною технікою та засобами візуалізації
  • комп’ютерна лабораторія, створена сумісно з компанією GlobalLogic Ukraine
  • клас дистанційного навчання MOODLE, створений за підтримкою Tempus-Tacis Joint European Project
  • навчально-дослідницькі лабораторії:
  • реконфігурованих систем «FPGA»
  • вбудованих систем «Embedded Systems»
  • мікропроцесорної техніки та систем реального часу
  • системного програмного забезпечення
  • інформаційних систем та баз даних.

Навчання здійснюється на професійному ліцензійному програмному забезпеченні. В університеті є вільний доступ до мережі Internet та внутрішньої університетської мережі, в якій кожному студенту виділено власний аккаунт.

Запрошуємо абітурієнтів вступати до Чорноморського національного університету імені Петра Могили на спеціальність 121 — «Інженерія програмного забезпечення».

Правила прийому 2021 року

Бакалаврат – 121 «Інженерія програмного забезпечення» Для вступу на перший курс на основі ПЗСО за спеціальністю 121 «Інженерія програмного забезпечення» необхідні сертифікати ЗНО з таких предметів:

  • Українська мова;
  • Математика;
  • іноземна мова або історія України, або біологія, або географія, або фізика, або хімія.

Для вступу на другий курс на основі ОКР «молодший спеціаліст» за спеціальністю 121 «Інженерія програмного забезпечення» необхідно скласти університетський фаховий іспит та 2 сертифікати ЗНО:

  • Українська мова;
  • Математика.

Магістратура – 121 «Інженерія програмного забезпечення»Вступ за результатами:

  • ЄВІ з іноземної мови;
  • ЄФВВ зі спеціальності.

Переваги навчання на спеціальності «Інженерія програмного забезпечення» ЧНУ імені Петра Могили:

  • денна та заочна форма навчання;
  • наявність бюджетних та контрактних місць;
  • військова кафедра;
  • міжнародні програми обміну студентів;
  • 3 власних гуртожитка;
  • окремий сучасний корпус факультету;
  • вивчення іноземних мов;

Кафедра інженерії програмного забезпечення ЧНУ імені Петра Могили м. Миколаїв, вул. 68 Десантників, 10, ауд. 2-302.
